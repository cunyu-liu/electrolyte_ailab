# 题目

分子式为  $\mathrm{C}_{8} \mathrm{H}_{14}$  的化合物 A 含有一个四级碳, 其与  $\mathrm{KMnO}_{4}$  加热回流后, 在反应溶液中仅检测到分子式为  $\mathrm{C}_{7} \mathrm{H}_{12} \mathrm{O}$  的产物。A 与氢溴酸作用得到 B , 而与氢溴酸在过氧化物的存在下作用得到异构体 C 。B 在乙醇钠-乙醇中得到 D , D 与  $\mathrm{BH}_{3}$  反应并进一步用乙酸处理后得到含有五元环结构的化合物 E 。D 可以使  $\mathrm{Br}_{2}$  褪色并生成 F , F 用乙醇钠-乙醇溶液处理生成两个异构体产物 G 和 H 。根据以上的信息推导所有物种的结构。

请根据你推断的结果，选择正确的选项。如果所有选项均错误，选择选项A：其他选项均不正确。

A. 其他选项均不正确  
B. 化合物 A 在核磁共振氢谱中有 5 种不同化学位移的氢  
C. 化合物  $\mathrm{A} \sim \mathrm{H}$  中有 4 种烃类物质  
D. 化合物  $\mathrm{A} \sim \mathrm{H}$  中有 2 对同分异构体  
E. 化合物  $\mathrm{G} / \mathrm{H}$  中与化合物  $\mathrm{A}$  中环上的取代基均完全相同  
化合物A中环上的不同种取代基位于相邻的C上

# 答案

正确答案: A

# 详细解析

解决这道题目，首先需要根据已知信息与 A 的反应性质，推断 A 的结构。

A 的分子式为  $\mathrm{C}_{8} \mathrm{H}_{14}$ , 其不饱和度  $\Omega = 2$  。

# CHECKPOINT

0.5 PTS

A 的不饱和度  $\Omega = 2$ 。

这说明其中含有不饱和  $\mathrm{C} = \mathrm{C}$  双键或环状结构。由于  $\mathbf{A}$  与  $\mathrm{KMnO}_4$  加热回流后，在反应溶液中仅检测到分子式为  $\mathrm{C}_7\mathrm{H}_{12}\mathrm{O}$  的产物。反应后的物种之比  $\mathbf{A}$  少了一个碳，这说明其中只含有一个不饱和  $\mathrm{C} = \mathrm{C}$  双键且为末端烯烃双键（ $[CX4][CX3]([CH2])[CX4]$ ），之后简记为  $[CC(= C)C]$ ），另一个不饱和度为环状结构贡献。

# CHECKPOINT

0.5 PTS

A 中含有一个末端烯烃双键、一个环状结构

A 与氢溴酸作用得到 B, 则后者三级碳正离子结合溴离子后的产物  $\left[{}^{1} C C\left(B r\right) \left(C\right) C l\right] 。$

# CHECKPOINT

0.5 PTS

B 中含有  $[JCC(Br)(C)C]$  片段

而与氢溴酸在过氧化物的存在下作用得到异构体 C，这是一个自由基反应，溴化氢加成双键的方向与离子反应相反，得到的产物应当为 `[JCC(CBr)C]`。

# CHECKPOINT

0.5 PTS

C 中含有`[]CC(CBr)C[]`片段

B 在乙醇钠-乙醇中得到 D，这是一步消除反应，需要注意的是此处并无法生成 A，而应该生成双键位置与 A 不同的同分异构体 D，故 D 的结构可以记为 `[JC = C(C)C]`。

# CHECKPOINT

0.5 PTS

D 中含有  $[ ] C = C (C) C [ ] ^ { \prime }$  片段

D 与  $\mathrm{BH}_{3}$  反应并进一步用乙酸处理后得到含有五元环结构的化合物  $\mathbf{E}$  。这一步反应是硼氢化还原反应，将不饱和的双键还原成饱和键，这显然没有发生成环，因此，此处所描述的“五元环”是底物一开始就有的，这也印证了“A 中一个环状结构”的设想，因此，此处的环状结构是五元环， $\mathbf{E}$  的结构包含`C1C(C)CCC1` 这样的环状结构，而  $\mathbf{A}$  则增加了环外的末端烯烃双键`C1C(=C)CCC1`。

# CHECKPOINT

1 PTS

E中含有`C1C(C)CCC1`结构、A中含有`C1C(=C)CCC1`结构

D 可以使  $\mathrm{Br}_{2}$  褪色并生成  $\mathbf{F}$ , 这一步反应发生了烯烃的二溴化, 根据之前的推论,  $\mathbf{F}$  中包含  $\mathrm{C} 1(\mathrm{Br}) \mathrm{C}(\mathrm{Br})(\mathrm{C}) \mathrm{CCC} 1$  结构。

# CHECKPOINT

0.5 PTS

F 中含有  $\mathrm{C} 1(\mathrm{Br}) \mathrm{C}(\mathrm{Br})(\mathrm{C}) \mathrm{CCC} 1$  结构

F 用乙醇钠-乙醇溶液处理生成两个异构体产物 G 和 H，这是一步消除反应，得到二烯烃，将 `C1(Br)C(Br)(C)CCC1` 改写为等价的 `[C:1]1[C:2]([Br:a])[C:3]([Br:b])[C:5][C:6]1` 表示，其中 `[C:1]` 上的氢能与 `[Br:a]` 发生消除且只有一种方式，这说明 `[C:1]` 并非题目中所指的 A 中的四级碳，同样的，`[C:4]/[C:5]` 的氢能与 `[Br:b]` 发生两种不同的消除方式得到两种异构体二烯烃，同时也说明 `[C:4]/[C:5]` 也不是四级碳，因此，只有 `[C:6]` 可以是四级碳，结合题目中 A 的分子式  $\mathrm{C}_{8}\mathrm{H}_{14}$ ，不难发现四级碳上相连了两个甲基，因此，A 的结构为 `C1C(=C)CC(C)(C)C1`。

# CHECKPOINT

3 PTS

A 的结构为 `C1C(=C)CC(C)(C)C1`

据此，可以得到：

- B 的结构为 `C1C(CBr)CC(C)(C)C1`  
- C 的结构为 `C1C(Br)(C)CC(C)(C)C1`

# CHECKPOINT

1 PTS

B 的结构为 `C1C(CBr)CC(C)(C)C1`、C 的结构为 `C1C(Br)(C)CC(C)(C)C1`

- D 的结构有两种可能性 `C1=C(C)CC(C)(C)C1` 或 `C1C(C)=CC(C)(C)C1`，若为后者，则其生成的 F 无法在乙醇钠的条件下发生消除反应得到两种不同的消除产物（只能消除一个溴、且有三种可能的产物），故其应该为 `C1=C(C)CC(C)(C)C1`

# CHECKPOINT

3 PTS

D 的结构为  $\mathrm{C} 1 = \mathrm{C}(\mathrm{C}) \mathrm{CC}(\mathrm{C})(\mathrm{C}) \mathrm{C} 1$

-  $\mathbf{E}$  的结构为  $\mathrm{C1C(C)CC(C)(C)C1}$  
- F 的结构为`C1(Br)C(Br)(C)CC(C)(C)C1`
