# 答案

正确答案: B

# 详细解析

催化剂 C 中配合物为中性分子，其中一个配体苯为中性分子，另一个配体为带两个负电荷的阴离子，因此两个镍应当总共带有两个正电荷，配合物结构对称，两个镍的价态应当相同，因此镍的化合价为 +1，选项A描述的镍的化合价为 +2 错误。

# CHECKPOINT

1 PTS

镍的化合价为  $+1$  ，选项A描述的镍的化合价为  $+2$  错误。

化合物B的作用为作为端基调节聚合度或聚合物分子量，与硅氧烷聚合中引入的三甲基氯硅烷的作用相似，因此选项B正确。
