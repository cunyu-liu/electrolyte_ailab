# 题目

卤素化学是元素化学中的重要部分，X元素是一种卤素。 $\mathbf{X}^{+}$ 简单离子往往被认为不能单独存在，但人们后来发现可以在以下条件获得包含正一价X元素的化合物A： $\mathbf{X}_{2} + \mathbf{B} + \mathbf{C} \rightarrow \mathbf{A}$ 。已知： $\mathbf{B}$ 为常见银源，且其阴离子不含卤素，为离子团； $\mathbf{C}$ 为有机合成中常见的碱，且具有一定亲核性，被认为能作为亲核性催化剂使用， $\mathbf{C}$ 的分子结构中有多中心离域键，只含有一个 $N$ 原子；生成 $\mathbf{A}$ 的反应过程中发生了氧化还原和酸碱络合反应。

根据上述信息，判断下列说法正确的选项。

A. X为溴元素，C与溴单质发生反应， $B r_{2}$ 作为亲电试剂被C的氮原子进攻。  
B. A 中正一价的 X 与一分子 C 结合, 从而得到稳定, 阴离子为硝酸根。  
C. X为碘原子，因为碘原子电负性相对小，与C以1比1的比例结合。  
D. B 为硝酸银, 其常被用于检验卤素阴离子, 在合成 A 的过程中硝酸根起到氧化剂的作用。  
E. A 中 X 原子与 C 分子的比例为 1 比 2 , 因为每个 C 分子中含有一个 N 原子, 所以 X 原子与 N 原子的比例也是为 1 比 2 。除了 A 外, 该反应还生成了含负一价 \mathsf{mathbf{\Phi}}\{\mathrm{X}\} \$\$ 的产物。  
F. A 中存在超价键作用。

# 答案

正确答案: F

# 详细解析

卤族元素电负性大，难以形成  $\mathbf{X}^{+}$ ，因此， $\mathbf{X}$  应该为半径最大，电负性最小的  $I$  元素；

# CHECKPOINT

1 PTS

X:I

B 为常见银源, 且阴离子不含有卤素, 所以应该为硝酸银  $\left(A g N O_{3}\right)$ , 亚硝酸银不常见;

# CHECKPOINT

1 PTS

B: AgNO3

C 为有机化学中常见碱, 能作为亲核催化剂, 且只有一个氮原子, 可以确定为吡啶  $\left(C_{5} H_{5} N\right)$ ;

# CHECKPOINT

1 PTS

C:  $C_5H_5N$

$I^{+}$  有一个空轨道和三个孤对电子，若为1比1配位则形成N-I单键，不稳定，能量太高；若为1比2配位，I的空轨道与两个吡啶上N的孤对电子形成3中心4电子键，相比一根单键更稳定。所以A为  $[I(C_{5}H_{5}N)_{2}]NO_{3}$

# CHECKPOINT

1 PTS

A:  $[I(C_{5}H_{5}N)_{2}]NO_{3}$

根据上述分析，各选项中说法正确的应该为F。

A中元素错误，且该反应历程复杂，不是简单的亲核进攻反应，错误。

B中X与吡啶比例错误。

C中X与吡啶比例错误。

D中硝酸根不作为氧化剂，错误。

E中硝酸根也有  $N$  原子，所以  $\mathbf{X}$  与  $N$  比例为1比3。
