# 题目

由三种主族金属元素形成的晶体化学式为  $X_{2}YZ(M < 200)$ ，其中  $X$  所占的质量分数为  $24.33\%$ ， $Y$  所占的质量分数为  $12.86\%$ 。

该晶体中  $Y$  、  $Z$  形成类似石墨的层状结构，每层中  $Y: Z = 1: 1$  ，上下两层不能完全重合，按照ABAB...进行排列，其中AB层  $Y$  原子是完全重合的，但  $Z$  完全不重合，  $X$  位于每个六边形中心上下方。

该晶体可以划分出以  $Y$  为顶点的六方晶胞，其晶胞参数  $a = 504.9 \, \text{pm}, c = 1010 \, \text{pm}$ ，每个六元环上下的  $X$  原子间的距离为  $343.8 \, \text{pm}$ 。

根据题意，在下列选项中选出正确的一项。

A. 晶体的组成部分中含有第三主族元素  
B. 晶体的组成部分中含有第五主族元素  
C. 晶体的组成部分中含有第六主族元素  
D. 晶胞中  $(\frac{1}{3}, \frac{2}{3}, 0.17)$  位置是  $Mg$  原子  
E. 晶胞中  $(\frac{1}{3}, \frac{2}{3}, \frac{1}{2})$  位置是  $I n$  原子  
F. 晶胞中  $(\frac{2}{3}, \frac{1}{3}, 0)$  位置是  $Sb$  原子  
G. 晶体密度是  $2.44 \mathrm{~g} / \mathrm{cm}^{3}$  
H. 晶体密度是  $2.95 \mathrm{~g} / \mathrm{cm}^{3}$

1. 晶体密度是  $2.82 g / c m^{3}$

# 答案

# 正确答案: I

# 详细解析

三種種類元素質素質的相對對應原質子質量測量儀之比較

[ X:Y:Z = (24.33\% / 2):12.86\% : (1 - 24.33\% - 12.86\%) = 12.17:12.86:62.81 ]，说明  $Z$  是较重的金属元素。由  $M < 200$  计算得到  $Z < 125.62$ 。

假设  $Z$  是  $\ln \left[\frac{114.82}{62.81\%}\right] = 182.8$  则

[ Y = 182.8 \times 12.86\% = 23.51, X = 182.8 \times 12.17\% = 22.25 ]，没有对应。

假设  $Z$  是  $\mathbf{Sn}$ ，计算  $M = \frac{118.71}{62.81\%} = 189.0$ ，则

[ Y = 189.0 \times 12.86\% = 24.31, X = 189.0 \times 12.17\% = 23.00 ]，对应  $Mg$ 、 $Na$ 。

假设  $Z$  是  $\sum_{b=1}^{20} S b$ , 计算  $M = \frac{121.76}{62.81\%} = 193.9$ , 则

$Y = 193.9 \times 12.86\% = 24.93, X = 193.9 \times 12.17\% = 23.60$  ，没有对应。

同样地，假设  $Z$  为其他主族金属元素，都没有对应的元素组合满足题目要求。于是晶体为  $Na_{2}MgSn$  。

# CHECKPOINT

2 PTS

晶体为  $N a_{2}M g S n$

六方晶胞的顶点为  $Mg$  原子，AB层的  $Sn$  原子不能重合，则一个六方晶胞含有一个A层一个B层， $Na$  原子处于层与层之间，总计包含两个  $Na_{2}MgSn$  结构单元。 $Mg$  原子对应的分数坐标为  $(0,0,0), (0,0,\frac{1}{2})$ 。

晶胞质量  $m = \frac{2\times 189}{6.023\times 10^{23}} = 6.28\times 10^{-22}g_{\circ}$

# CHECKPOINT

0.5 PTS

晶胞质量  $m = 6.28 \times 10^{-22} g$

晶胞体积  $V = \frac{\sqrt{3}}{2} a^{2}c = \frac{\sqrt{3}}{2}\times 504.9^{2}\times 1010 = 2.23\times 10^{-22}cm^{3}$

# CHECKPOINT

0.5 PTS

晶胞体积  $V = 2.23 \times 10^{-22} \mathrm{~cm}^{3}$

晶体密度  $\rho = \frac{m}{V} = \frac{6.28 \times 10^{-22}}{2.23 \times 10^{-22}} = 2.82 \, \text{g/cm}^3$ 。

# CHECKPOINT

1 PTS

晶体密度  $\rho = 2.82 g / c m^{3}$
