# 题目

某化合物X仅含A和B两种元素。

在高压下， $\mathbf{X}$  形成了一种原子晶体。此晶体为正交晶系，其中  $\mathbf{A}$  均为四配位， $\mathbf{B}$  均为二配位， $a = 622 \mathrm{pm}, b = 435 \mathrm{pm}, c = 607 \mathrm{pm}, \rho = 3.56 \mathrm{g} \cdot \mathrm{cm}^{-3}, Z = 8$ 。

X在低温高压下可形成一种立方晶系的分子晶体，  $Z = 4$  ，分子无法自由转动。

在高压下，X还能形成一种四方晶系的分子晶体，晶胞参数  $a = 354\mathrm{pm},\mathrm{c} = 413\mathrm{pm},\mathrm{Z} = 2$  。在一个正当晶胞中，若以A原子为晶胞顶点，则其中两个B的坐标分别为(0.266,0.266,0)和(0.766,0.234,0.5)。

请选出以下命题中所有正确的选项：

1. X 分子有对称中心  
2.  $\mathbf{X}$  的立方晶体的点阵类型是简单立方  
3.  $\mathbf{X}$  的四方晶体有对称中心  
4. X 的四方晶体有穿过 B 原子的二重旋转轴  
5. X 的四方晶体没有不穿过 A 原子的二重旋转轴

A. 1,3,4  
B. 1,3,5  
C. 1,4,5  
D. 2,3,4  
E. 2,3,5

F. 2,4,5  
G. 1,2,3,4  
H. 1,2,3,5  
1,2,4,5  
J. 1,3,4,5  
K.  $1,2,3,4,5$

# 答案

正确答案: G

# 详细解析

首先推断  $\mathbf{X}$  的化学式：

晶胞体积  $V = a \times b \times c = 622\mathrm{pm} \times 435\mathrm{pm} \times 607\mathrm{pm} = (1.643 \times 10^{8})\mathrm{pm}^{3} \approx (1.643 \times 10^{-22})\mathrm{cm}^{3}$

X 的摩尔质量  $M = \frac{\rho \times V \times N_{A}}{Z} = \frac{3.56 \times 1.643 \times 10^{-22} \times 6.022 \times 10^{23}}{8} \approx 44.0 \mathrm{~g} \cdot \mathrm{mol}^{-1}$

# CHECKPOINT

2 PTS

X的摩尔质量  $M\approx 44.0\mathrm{g}\cdot \mathrm{mol}^{-1}$

由于化学键的总数是守恒的，A对B的配位数等于B对A的配位数，故有  $4\times n_{A} = 2\times n_{B}$  。

所以A和B的原子个数比为1:2

A和B的原子个数比为1:2

符合以上条件的化合物可能是  $\mathrm{CO}_{2}$  或  $\mathrm{N}_{2} \mathrm{O}$ , 但  $N_{2} O$  难以实现氧元素的四配位, 所以  $\mathbf{X}$  的化学式是  $\mathrm{CO}_{2}$ ,

A是C，B是O

# CHECKPOINT

1 PTS

X是  $\mathrm{CO}_{2}$  ，A是C，B是O

$\mathrm{CO}_{2}$  分子具有对称中心，所以命题1正确。

# CHECKPOINT

1 PTS

$\mathrm{CO}_{2}$  分子具有对称中心, 命题1正确

$\mathrm{CO}_{2}$  形成的立方晶系的分子晶体，其正当晶胞包含4个  $\mathrm{CO}_{2}$  分子，如果为面心立方点阵，则4个  $\mathrm{CO}_{2}$  分子的空间取向必须一致，这将导致立方晶系对称性的丧失。所以，在题设条件下，X 的立方晶体只能是简单立方点阵。命题2正确。

# CHECKPOINT

1 PTS

X的立方晶体是简单立方点阵，命题2正确

接下来分析X的四方晶体。可以借助四方晶系的对称性和  $\mathrm{CO}_{2}$  的分子对称性，得到未给出的原子坐标。

设  $\mathbf{C1}(0,0,0)$  ，O1(0.266,0.266,0)，O2(0.766,0.234,0.5)

C1和O1属于同一个分子，则可得O3(0.734,0.734,0)

根据四方晶系的特征对称元素，该晶体必须具有  $z$  方向的四重旋转轴、反轴或螺旋轴。  $\mathrm{CO}_{2}$  分子在  $z$  方向不满足四重旋转轴或反轴的要求，则只能是  $z$  方向的  $4_{2}$  螺旋轴。由此可以确定，C2(0.5,0.5,0.5),O4(0.234,0.766,0.5)

C2(0.5, 0.5, 0.5), O3(0.734, 0.734, 0), O4(0.234, 0.766, 0.5)

接着分析对称元素，该晶体中所有C原子的位置都是对称中心，所以命题3正确。
