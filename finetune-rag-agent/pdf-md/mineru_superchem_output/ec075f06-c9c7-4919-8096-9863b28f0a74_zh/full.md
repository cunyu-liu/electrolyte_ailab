# 题目

# 答案

正确答案: S

# 详细解析

由M的质量分数和失重信息，可知M在氧化物B中的质量分数为  $\frac{0.2630}{1 - 0.5031 - 0.1677} = 0.7989$  ，因此B中氧元素的质量分数为  $20.11\%$

若设B化学式为  $\mathrm{MO_a}$  ，则M的原子量为  $16\mathrm{a}\times 0.7989 / 0.2011$

# CHECKPOINT

1 PTS

设  $\mathrm{B}$  化学式为  $\mathrm{MO}_{\mathrm{a}}$  ，则M的原子量为  $16 \mathrm{a} \times 0.7989 / 0.2011$

枚举可知当  $\mathrm{a} = 1$  时，  $\mathrm{M_M} = 63.56\mathrm{g}\cdot \mathrm{mol}^{-1}$  ，对应于Cu

# CHECKPOINT

1 PTS

M为Cu

代入  $\mathbf{X}$  中  $\mathbf{M}$  的质量分数可知  $\mathbf{X}$  的分子量为241.63，扣除  $\mathbf{M}$  后剩余分子量约为178.1。由于HY为一元酸，联系铜元素的常见氧化态可以推知  $\mathrm{x} = 2$ ；此时再由  $\mathbf{M}$  可以溶于HY得到  $\mathbf{X}$ ，且失重时放出有颜色的刺激性气体，可以大胆猜测HY为硝酸。代入剩余分子量，扣除两个硝酸根的式量，可知水分子数量恰好为3，同时也验证了HY为硝酸的猜想。因此  $\mathbf{X}$  为  $\mathrm{Cu(NO_3)_2\cdot 3H_2O}$

# CHECKPOINT

2 PTS

X为  $\mathrm{Cu(NO_3)_2\cdot 3H_2O}$

由于失重过程中  $\mathrm{Cu}$  氧化态没有发生变化，因此  $\mathbf{X}$  的分解可以视为铜离子发生水解、加热使  $\mathbf{X}$  脱去了硝酸分子和水分子，而  $\mathbf{A}$  则是一种碱式硝酸铜。

由失重质量分数，第一次失重分子量为  $241.63 \times 0.5031 = 121.56$  ，第二次为  $241.63 \times 0.1677 = 40.52$  （如果对数据足够敏锐的话会发现二者之比非常接近于3:1）。

硝酸的分子量为63.018，水分子为18.016，在1个  $\mathrm{Cu}$  的情况下无解，因此考虑扩倍为2个  $\mathrm{Cu}$ ，失重243.12，对应于  $3\mathrm{HNO}_3 + 3\mathrm{H}_2\mathrm{O}$ ；而第二次则是  $\mathrm{HNO}_3 + \mathrm{H}_2\mathrm{O}$ 。因此，A为  $\mathrm{Cu}_2(\mathrm{OH})_3\mathrm{NO}_3$

# CHECKPOINT

2 PTS

A 为  $\mathrm{Cu}_{2}(\mathrm{OH})_{3} \mathrm{NO}_{3}$

第一次失重:  $8\left[\mathrm{Cu}\left(\mathrm{NO}_{3}\right)_{2} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right] = 4 \mathrm{Cu}_{2}(\mathrm{OH})_{3} \mathrm{NO}_{3} + 3 \mathrm{O}_{2} + 12 \mathrm{NO}_{2} + 18 \mathrm{H}_{2} \mathrm{O}$

# CHECKPOINT

第一次失重:  $8\left[\mathrm{Cu}\left(\mathrm{NO}_{3}\right)_{2} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right] = 4 \mathrm{Cu}_{2}(\mathrm{OH})_{3} \mathrm{NO}_{3} + 3 \mathrm{O}_{2} + 12 \mathrm{NO}_{2} + 18 \mathrm{H}_{2} \mathrm{O}$

第二次失重：  $4\mathrm{Cu}_{2}(\mathrm{OH})_{3}\mathrm{NO}_{3} = 8\mathrm{CuO} + \mathrm{O}_{2} + 4\mathrm{NO}_{2} + 6\mathrm{H}_{2}\mathrm{O}$

$\mathrm{a} = 45$  ，  $\mathrm{b} = 23$  ，  $\mathrm{a / b} = 1.96$  ，选S
