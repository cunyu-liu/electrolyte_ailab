# Answer

Correct Answer: K

# Detailed Explanation

First, consider the number of isomers when the four halogen atoms are identical:

(1) All  $4 \mathrm{X}$  on the same face: 1,2,3,4- 1 type  
(2) Three X on one face: cases include 1,2,3,5- / 1,2,3,6- / 1,2,4,5- — 3 types  
(3) Two X on the same face: cases include 1,2,5,6- / 1,3,5,7- — 2 types

Thus, there are a total of 6 possible arrangements.
