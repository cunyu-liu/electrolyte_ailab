# 题目

碘量法是氧化还原滴定中重要的实验方法。实验室试用碘量法测定  $I_{2} + I^{-} = I_{3}^{-}$  反应的平衡常数，方法如下所示：称取三份  $0.5\mathrm{g}$  研细的碘于编号为1、2、3的三个磨口锥形瓶中。1号瓶中加入 $80.0\mathrm{mL}$  浓度为  $c_{0}(\mathrm{mol}\cdot \mathrm{L}^{-1})$  的KI溶液；2号瓶中加入  $80.0\mathrm{mL}$  浓度为  $2c_{0}(\mathrm{mol}\cdot \mathrm{L}^{-1})$  的KI溶液；3号瓶中加入  $200.0\mathrm{mL}$  去离子水。加入磁力搅拌子，塞紧塞子后室温下搅拌反应30分钟。反应结束后，从三个瓶中取出溶液，离心得到上层清液。各取  $V_{s}(\mathrm{mL})$  离心后的1号和2号溶液，加水稀释至 $50~\mathrm{mL}$  。用浓度为  $c(\mathrm{mol}\cdot \mathrm{L}^{-1})$  的  $\mathrm{Na_2S_2O_3}$  标准溶液滴定，消耗  $\mathrm{Na_2S_2O_3}$  标准溶液的体积分别为 $V_{1}(\mathrm{mL})$  和  $V_{2}(\mathrm{mL})$  。然后取  $5V_{s}(\mathrm{mL})$  离心后的3号溶液，用相同浓度的  $\mathrm{Na_2S_2O_3}$  标准溶液滴定，消耗  $\mathrm{Na_2S_2O_3}$  标准溶液的体积为  $V_{0}(\mathrm{mL})$  。

用题中所给的  $c, c_0, V_0, V_1, V_s$  可以计算题目所给反应的平衡常数  $K$ 。

下列操作失误会使平衡常数  $K$  的测定结果变大的有哪些？

1. 三份碘的质量稍有不同。  
2. 搅拌结束后3号瓶澄清透明。  
3. 取1号溶液时不慎将沉在底部的固体吸入移液管。  
4. 配置KI溶液时，称量完毕后不慎遗撒少量KI固体。  
5.  $\mathrm{Na}_{2} \mathrm{~S}_{2} \mathrm{O}_{3}$  标准溶液标定完后放置过久。

A. 1,2,3  
B. 3,5  
C. 1,4  
D. 2,3,4  
E. 2,3,5

F. 2,3  
G. 2,3,4,5  
H. 3,4

# 答案

正确答案: E

# 详细解析

锥形瓶1和3的数据合起来可以计算平衡常数  $K$  。锥形瓶3可得饱和  $I_{2}$  的浓度，结合锥形瓶1的数据可得反应 $I_{2} + I^{-} = I_{3}^{-}$  的平衡常数。推导得  $K = (5V_{1} - V_{0})*V_{s} / [(c_{0}V_{s} - 0.5cV_{1} + 0.1cV_{0})*V_{0}]$

分析各操作失误对  $K$  的影响：

1. 实验中碘是过量的，所以碘的质量对  $K$  没有影响。

# CHECKPOINT

1 PTS

碘过量，所以碘的质量对  $K$  没有影响。

2. 搅拌结束后3号瓶澄清透明，说明碘添加量不够，测得的饱和碘浓度偏低，导致平衡常数  $K$  偏大。

# CHECKPOINT

1 PTS

失误2使测得的饱和碘浓度偏低，  $K$  偏大。

3. 取1号溶液时不慎将沉在底部的固体吸入移液管，则测得的  $I_{3}^{-}$ 浓度偏大，导致平衡常数  $K$  偏大。

# CHECKPOINT

1 PTS

失误3使测得的  $I_{3}^{-}$  浓度偏大， $K$  偏大。

4. 配置KI溶液时，称量完毕后不慎遗撒少量KI固体，会导致实际的KI浓度小于  $c_{0}$ ，即计算中使用的  $c_{0}$  偏大，结合  $K$  的表达式可知  $c_{0}$  偏大会导致  $K$  偏小。

# CHECKPOINT

1 PTS

失误4使  $c_{0}$  偏大， $K$  偏小。

5.  $\mathrm{Na}_{2} \mathrm{~S}_{2} \mathrm{O}_{3}$  标准溶液标定完后放置过久浓度会变小, 即计算中使用的  $c$  偏大, 结合  $K$  的表达式可知  $c$  偏大会导致  $K$  偏大。

# CHECKPOINT

1 PTS

失误5使  $c$  偏大,  $K$  偏大

所以使  $K$  偏大的操作有2、3和5。
