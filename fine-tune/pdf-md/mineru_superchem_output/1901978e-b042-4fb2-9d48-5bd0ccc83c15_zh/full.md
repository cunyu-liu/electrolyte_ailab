# 答案

正确答案: G

# 详细解析

在光照下, 烷基碘化物的  $\mathrm{C}-\mathrm{I}$  键均裂, 产生引发自由基  $R C F_{2} .$  (即  $\mathbf{X}$  ) 和一个碘自由基,  $\mathbf{X}$  进攻1,1-二氟乙烯引发聚合反应。随后可发生链转移, 聚合物自由基进攻  $R C F_{2} I$  攫取碘原子, 产生新的  $\mathbf{X}$ ; 或聚合物自由基与一分子碘自由基反应, 终止反应。
