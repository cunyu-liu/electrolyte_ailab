# 题目

同位素标记是经典的用于研究生物体内的代谢过程的方法。我们考虑用不同  $^{13}C$  标记的葡萄糖在氧气充足的条件下培养哺乳动物细胞，并分别添加不同的外源分子调控细胞代谢。以下是关于相应实验方法和预期结果的描述，请选出其中正确的一项。

A. 1用  ${ }^{13} C$  标记C4位的葡萄糖培养细胞, 不加任何其他外源小分子, 则丙酮酸中最先被同位素标记的碳为酮炭基碳。  
B. 用碘乙酸预处理细胞后, 撤去培养液。用  $^{13} C$  标记C6位的葡萄糖培养细胞, 则最终同位素标记停留在3-磷酸甘油醛的醛基碳或磷酸二羟基丙酮的磷酸基碳。  
C. 用砷酸钠预处理细胞后, 撤去培养液。用  $^{13} C$  标记  $\mathrm{C} 3$  位的葡萄糖培养细胞, 不加任何其他外源小分子, 则丙酮酸不会被同位素标记。  
D. 用  ${ }^{13} \mathrm{C}$  标记  $\mathrm{C} 6$  位的葡萄糖培养细胞, 并在培养液中加入一定浓度的氟化钠, 则最终同位素标记集中在磷酸烯醇式丙酮酸的亚甲基碳。  
E. 用  ${ }^{13} C$  标记C4位的葡萄糖培养细胞后，在短时间内收集细胞并分离细胞内的饱和脂肪酸，在这些脂肪酸中能检测到  ${ }^{13} C$  的同位素标记。  
F. 用  ${ }^{13} C$  标记C1位的葡萄糖培养细胞后, 分离细胞内的酮体并检测同位素标记, 则同位素标记应当出现在3-羟基丁酸、乙酰乙酸的甲基与亚甲基, 和丙酮的酮炭基碳。  
G. 用  $^{13} C$  标记C6位的葡萄糖培养细胞, 并在培养液中加入高浓度的氟乙酸, 该处理不会影响草酰乙酸被检测到同位素标记。  
H. 用  ${ }^{13} \mathrm{C}$  标记  $\mathrm{C} 3$  位的葡萄糖培养细胞, 最少需要经过 1 轮柠檬酸循环, 就可以在细胞代谢产生的二氧化碳中检测到同位素标记。

1. 用  $^{13}C$  标记C6位的葡萄糖培养细胞, 最少需要经过3轮柠檬酸循环, 才能在柠檬酸循环中产生有同位素标记的二氧化碳。  
J. 以上说法均不正确

# 答案

# 正确答案: I

# 详细解析

首先考察糖酵解过程，糖酵解中葡萄糖的C1-C6在生成1,6-二磷酸果糖为止都保持不变。随后1,6-二磷酸果糖发生裂解，1,2,3号碳进入磷酸二羟基丙酮，1号碳连磷酸基，2号碳为羰基，3号碳连羟基；4,5,6号碳进入3-磷酸甘油醛，4号碳为醛基，5号碳连羟基，6号碳连磷酸基.在磷酸丙糖异构酶的作用下，磷酸二羟丙酮异构化位3-磷酸甘油醛，该过程不涉及磷酸基的迁移，因此1号碳为磷酸基，2号碳连羟基，3号碳为醛基。故葡萄糖标记中，在形成3-磷酸甘油醛之后的过程里，1/6号，2/5号，3/4号组内是两两等价的。

# CHECKPOINT

1 PTS

同位素标记的葡萄糖中，标记可以分为1/6，2/5，3/4三种情况，组内两种标记是等价的。

首先考察3/4组，对应选项为A、C、E、H.3-磷酸甘油醛的醛基在丙酮酸中对应羧基，5号碳对应酮羰基，6号碳对应甲基，因此丙酮酸中的同位素出现在羧基。A选项错误。

# CHECKPOINT

1 PTS

标记葡萄糖的C4则同位素出现在丙酮酸的羧基，A错误

砷酸钠是3-磷酸甘油醛氧化的抑制剂，能够解除氧化和磷酸化的偶联作用，但是为可逆抑制，撤去培养液后，糖酵解恢复正常，故C选项错误。

# CHECKPOINT

1 PTS

砷酸钠的抑制是可逆的，在撤去后糖酵解能恢复正常，C错误

# CHECKPOINT

1 PTS

同位素会首先出现在丙酮酸的羧基

丙酮酸转化成乙酰辅酶A，羧基以二氧化碳的形式离去，故同位素不会出现在下游的代谢过程中，选项E、H均错误。

# CHECKPOINT

1 PTS

丙酮酸羧基的同位素标记不会进入乙酰辅酶A，故不会出现在细胞合成的饱和脂肪酸中，E错误

# CHECKPOINT

1 PTS

不需要进入柠檬酸循环，就能通过乙酰辅酶A的合成产生带有同位素标记的二氧化碳，H错误

下面考察1/6组，对应描述为2，4，6，7，9，10.碘乙酸为GAPDH的不可逆抑制剂，撤去培养液不能逆转抑制效果。最终糖酵解终止在3-磷酸甘油醛（磷酸二羟基丙酮）处，同位素标记的是与磷酸基相连的碳，选项B错误。

# CHECKPOINT

1 PTS

碘乙酸是GAPDH的不可逆抑制剂

# CHECKPOINT

1 PTS

同位素标记在两种化合物中都位于与磷酸基相连的碳，B错误

氟化钠是丙酮酸激酶和烯醇化酶的抑制剂，因此糖酵解会停留在磷酸烯醇式丙酮酸之前的2-磷酸甘油酸处，而非集中在磷酸烯醇式丙酮酸。同位素标记的是亚甲基碳，选项D错误。

# CHECKPOINT

1 PTS

氟化钠抑制丙酮酸激酶和烯醇化酶

# CHECKPOINT

1 PTS

同位素标记主要停留在2-磷酸甘油酸，D错误

乙酰辅酶A中，甲基碳带有同位素标记，此后进入酮体合成途径，3分子乙酰辅酶A合成HMG-CoA，其中存在同位素标记的是硫酯α位碳、β位上取代的甲基的碳、和羧基α位碳。随后HMG-CoA裂解产生乙酰乙酸，同位素标记出现在甲基和亚甲基。乙酰乙酸脱羧产生丙酮，同位素标记位于两个甲基；乙酰乙酸还原为3-羟基丁酸，同位素位置不变，仍然在甲基和亚甲基。选项F错误。

# CHECKPOINT

1 PTS

甲基标记的乙酰辅酶A合成的酮体分子中，同位素均位于甲基或者亚甲基处，F错误

氟乙酸转化为氟乙酰CoA后，可以与草酰乙酸反应为氟柠檬酸，并抑制柠檬酸合酶，从而阻断柠檬酸循环。考虑到同位素标记的葡萄糖需要糖酵解产生乙酰辅酶A，而大量氟乙酸在一开始即能阻断柠檬酸循环，因此含有同位素的乙酰辅酶A几乎不能进入柠檬酸循环，故草酰乙酸中几乎不能检测到同位素的标记。选项G错误。

# CHECKPOINT

1 PTS

氟乙酸转化成的氟柠檬酸会抑制柠檬酸合酶

# CHECKPOINT

1 PTS

柠檬酸循环被阻断，草酰乙酸中不会检测到同位素标记

下面考察柠檬酸循环的同位素传递，乙酰辅酶A的甲基上有同位素标记，进入三羧酸循环后会出现在琥珀酸的其中一个亚甲基上。注意到琥珀酸是完全对称的分子，在水化为苹果酸并氧化为草酰乙酸后，同位素标记可能是在草酰乙酸的羰基或者亚甲基上。若亚甲基带有同位素，则在下一轮循环中，同位素会出现在草酰乙酸的羧基上。若羰基上带有同位素，则在下一轮循环中，同位素会出现在柠檬酸羟基碳上，并回到琥珀酸的亚甲基，然后再回到草酰乙酸的羰基或者亚甲基上。下面仅考虑草酰乙酸的羧基的同位素标记，则其中一个羧基会在  $\alpha$ -酮戊二酸转化为琥珀酰辅酶A的过程中转变为二氧化碳。故选项I是正确的。

# CHECKPOINT

1 PTS

最少需要三轮柠檬酸循环：乙酰辅酶A  $\rightarrow$  草酰乙酸的亚甲基  $\rightarrow$  草酰乙酸的羧基  $\rightarrow$  二氧化碳，就可以把同位素传递到二氧化碳上；I选项正确
