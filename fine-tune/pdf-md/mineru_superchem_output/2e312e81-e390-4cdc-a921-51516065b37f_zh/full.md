# 题目

黄色固体 A 由金属元素 X 和非金属元素 Y 两种元素构成, 是 X 元素的重要矿物之一。A 在空气中煅烧可生成固体 B 与无色气体 C, B 中含 X 元素  $69.94\%$  。C 与  $\mathrm{Cl}_2$  反应得到无色发烟液体 D, D 极易水解生成 E 和 F, C 可以在催化剂加热作用下和  $\mathrm{O}_2$  反应生成 G, G 可溶于水也得到 E。

C 是一种优良的 Lewis 酸,科研人员日前发现将  $0.5 \mathrm{~mL}$  无水 C 在低温下与  $0.02 \mathrm{~g}$

某金属叠氮盐 I 反应可以得到加成物 H, H 中氮的含量为  $13.87\%$ , 若忽略 H 中 C 的取向, 则其阴离子对称性与水相同。

下列说法正确的是:

A. 其他选项均不正确  
B. A是1:2型离子化合物  
C.  $\mathbf{X}$  元素可以形成的最高氧化态为  $+4$  
D. D为正四面体构型分子  
E. G 具有一定的还原性  
F. I中氮的质量分数约为  $28\%$  
G. H中阳离子的质量分数为  $43.9\%$  
H. H为2: 1型离子化合物。

# 答案

正确答案: G

# 详细解析

B由A煅烧得到，可以判断B为X元素的稳定氧化物；设B的化学式为  $\mathrm{XO_n}$  ，含X元素  $69.94\%$  。

# CHECKPOINT

1 PTS

B为X元素的稳定氧化物

若为XO，X的相对原子质量为  $16 / (1 - 0.6994) - 16 = 37.2$  ，无合理金属。

若为  $\mathbf{X}_2\mathrm{O}_3$  ，X的相对原子质量为  $(16\times 3 / (1 - 0.6994) - 16\times 3) / 2 = 55.9$  ，为Fe

若为  $\mathbf{XO}_2$  ，  $\mathbf{X}$  的相对原子质量为  $16\times 2 / (1 - 0.6994) - 16\times 2 = 74.5$  ，无合理金属。

其他氧化物类型也均无合理结果；因此X为Fe，B为  $\mathrm{Fe_2O_3}$  。Fe存在  $+6$  价，选项C错误。
