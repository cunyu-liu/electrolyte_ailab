# 题目

下面有两个反应

反应1:

![](images/bbce7f848901d881d260f9b510b65b15b4099c4e4097ce51b475b22ac835655d.jpg)  
CC1(CC(C2=C(SCCCC([R1])C2)C1)=O)C> [R]S.[DBU]>，随后加入TFA得到产物

反应2:

![](images/6579d45df52099c9b52b2c2a8377949716a3521620ed394d4c6e30644de95791.jpg)  
C=C1C(OCCCCCCCCCCOC(CSC1C2=CC=CC=C2)=O)=O> [1%DBU].[DMF].[r.t.]>

下面说法正确的是？

A. 反应1存在关键中间体:

![](images/e3d16974244f523bbcb270507fcaa73e99a0269524c1cbae813aab7e533f8e89.jpg)  
CC1(CC(C2C(SCCCC([R1])C2)(S[R])C1)=O)C

B. 反应1存在关键中间体:

![](images/43d6db9c06160a85c6a8bf5c1fc4d689f37fa7757b5b19c499338ce91df01548.jpg)  
CC1(CC(C(CC([R1])CCCS[R])C([S-])C1)=O)C

C. 反应1的可能存在副产物:

![](images/11b4f16f4ea0ebf5b74cb209e6e6367ff55c61577cfb160408f983c4396578a6.jpg)

CC1(CC(C(CC([R1])CCCS)C(S[R])C1)=O)C

D. 反应1的产物端基不确定  
E. 反应2的产物为:

![](images/a7627a9e22a8d90cbb67209d50421bd36612f447c2b39f37867853e46a1e29dc.jpg)

$\mathrm{O = C(OCCCCCOC(C[X2]) = O)C(CC1 = CC = CC = C1)CS[X1]}$  ，其中X1和X2是重复单元的连接位点

F. 反应2的产物为:

![](images/8e4ac371c7e9bf46051cbaf0a050e92c6bf7205bd0329060bac444efa94aa7ff.jpg)

$\mathrm{O = C(OCCCCCOC(C[X2]) = O) / C(CS[X1]) = C / C1 = CC = CC = C1}$ , 其中X1和X2是重复单元的连接位点

G. 反应2的端基是确定的  
H. 其他选项均不正确

# 答案

正确答案: F

# 详细解析

反应1为开环聚合反应，DBU作为强碱使RSH去质子化生成硫醇负离子对底物alpha-beta不饱和酮进行1,4-加成得到烯醇负离子中间体1，该中间体开环得到硫醇负离子中间体2，随后中间体2重复该过程完成聚合，TFA用于淬灭反应，反应1产物的端基为RS和H。
