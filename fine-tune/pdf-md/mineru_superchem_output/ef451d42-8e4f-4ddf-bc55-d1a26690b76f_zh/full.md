# 题目

化合物A可由2,4-二磺酸基苯酚与浓硝酸反应制得。1873年，Hermann Sprengel发现A可作起爆药，此后的甲午战争，日俄战争及第一次世界大战中，A都被广泛地使用。1848年，苏格兰化学家John Stermhouse混合次氯酸钙与少量A制得了B，B在一战中被德军用作催泪剂，现在其被用作杀虫剂。B不含  $H$  原子或  $C - C$  键，其中  $\omega_{C} = 7.31\%$  。用  $Sn$  在盐酸中还原B，得到另一种曾被德军制备但未应用的糜烂性毒剂C，相对分子质量小于150；C亦可由雷管的有效物质D与氯气在盐酸中反应得到；C中所有原子共面，不存在电荷分离。

电解  $\mathbf{B}$  的甲醇-硫酸溶液，在阴极得到化合物  $\mathbf{E}$ ； $\mathbf{E}$  也可用亚硝酰氯与三氯甲基亚磺酸钠反应制备。在  $\mathbf{E}$  的熔点附近加热， $\mathbf{E}$  分解为等摩尔的  $\mathbf{B}$ 、 $\mathbf{F}$  和亚硝酰氯；而在  $120^{\circ} \mathrm{C}$  加热时，则生成摩尔比  $1:1:2$  的常见溶剂  $\mathbf{G}$ 、化合物  $\mathbf{H}$  和气体  $\mathbf{I}$ 。 $\mathbf{F}$  与  $\mathbf{H}$  的结构相似，分子量相差  $16$ 。

推断A至I的化学式，以下说法中错误的有：

1. 由  $\mathbf{A}$  和次氯酸钙生成  $\mathbf{B}$  的反应方程式中, 含  $C a$  的产物有2种  
2.1当量B生成C需要消耗2当量Sn  
3.1当量D生成2当量C  
4.1当量B生成E消耗2当量电子  
5. H中不存在电荷分离  
6. F的结构可以视为C中一个取代基被替换后的结构

# 答案

正确答案: J

# 详细解析

显然  $\mathbf{A}$  是三硝基苯酚（ $C_{6}H_{2}(OH)(NO_{3})_{3}$ ）。根据  $\mathbf{B}$  不含  $H$  原子或  $C - C$  键，判断  $\mathbf{B}$  可能只含一个  $C$ ， $\mathbf{B}$  的相对分子质量为  $12.01 / 0.0731 = 164.3$ ，除去  $C$  的质量后，对应3个  $Cl$  和1个  $NO_{2}$ ，因此  $\mathbf{B} = CCl_{3}NO_{2}$ 。

# CHECKPOINT

1 PTS

A是三硝基苯酚  $(C_{6}H_{2}(OH)(NO_{3})_{3})$

# CHECKPOINT

1 PTS

$$
\mathbf {B} = C C l _ {3} N O _ {2}
$$

配平由  $\mathbf{A}$  生成  $\mathbf{B}$  的方程式如下：

$$
2 C _ {6} H _ {2} (O H) \left(N O _ {2}\right) _ {3} + 1 1 C a (O C l) _ {2} = 6 C C l _ {3} N O _ {2} + 6 C a C O _ {3} + 3 C a (O H) _ {2} + 2 C a C l _ {2}
$$

# CHECKPOINT

1 PTS

$$
2 C _ {6} H _ {2} (O H) \left(N O _ {2}\right) _ {3} + 1 1 C a (O C l) _ {2} = 6 C C l _ {3} N O _ {2} + 6 C a C O _ {3} + 3 C a (O H) _ {2} + 2 C a C l _ {2}
$$

有3种含  $C a$  产物，说法1错误。

C 可由 B 在盐酸中被  $S_{n}$  还原得到, B 被完全还原的产物可能是  $C C l_{3} N H_{2}$ , 但是不是平面化合物, 因此 C 可能是 B 部分还原的产物。C 所有原子共平面, 因此很可能形成了  $C = N$  双键, 想到 C 的结构可能为  $C l_{2} C N(OH)$  或者为  $C l_{2} C N H$ , 与已知信息没有矛盾之处。

# CHECKPOINT

1 PTS

$\mathbf{C} = Cl_2CN(OH)$  或  $Cl_{2}CNH$

雷管的有效物质  $\mathbf{D}$  可能是叠氮化氢或者雷酸汞，结合  $\mathbf{C}$  的结构，可以判断后者更符合，即  $\mathbf{D} = Hg(CNO)_2$

# CHECKPOINT

1 PTS

$$
\mathbf {D} = H g (C N O) _ {2}
$$

D 在盐酸中与  $Cl_2$  反应生成 C，若 C 为  $Cl_2CNH$ ，很难解释雷酸汞中的氧原子去向，因此 C 为  $\mathbf{C} = Cl_2CN(OH)$ 。

# CHECKPOINT

1 PTS

排除  $Cl_{2}CNH$  ，  $\mathbf{C}$  为  $\mathbf{C} = Cl_{2}CN(OH)$  。

用  $S_{n}$  在盐酸中还原  $\mathbf{B}$  得到  $\mathbf{C}$  的方程式如下：

$$
C C l _ {3} N O _ {2} + 2 S n + 3 H C l = C l _ {2} C N (O H) + 2 S n C l _ {2} + H _ {2} O
$$

# CHECKPOINT

1 PTS

$$
C C l _ {3} N O _ {2} + 2 S n + 3 H C l = C l _ {2} C N (O H) + 2 S n C l _ {2} + H _ {2} O
$$

1当量B生成C需要消耗2当量  $Sn$  ，说法2正确。

D 在盐酸中与  $C l_{2}$  反应生成 C 的方程式如下:

$$
H g (C N O) _ {2} + 2 C l _ {2} + 2 H C l = 2 C l _ {2} C N (O H) + H g C l _ {2}
$$

# CHECKPOINT

1 PTS

$$
H g (C N O) _ {2} + 2 C l _ {2} + 2 H C l = 2 C l _ {2} C N (O H) + H g C l _ {2}
$$

1当量D生成2当量C，说法3正确。

三氯甲基亚磺酸钠可以做三氯甲基化试剂, 由  $\mathbf{E}$  可用亚硝酰氯与三氯甲基亚磺酸钠反应制备, 得  $\mathbf{E}$  的化学式为  $C l_{3} C N O$

# CHECKPOINT

1 PTS

$$
\mathbf {E} = C l _ {3} C N O
$$

1当量B还原为E消耗2当量电子，说法4正确。

$\mathbf{E}$  在熔点附近加热的反应方程式为：

$$
n C l _ {3} C N O = C C l _ {3} N O _ {2} + \mathbf {F} + N O C l
$$

由产物中  $O$  的个数可知  $n \geq 3$  。当  $n = 3$  时， $\mathbf{F} = C_2C l_5N$ ，结合  $\mathbf{E}$  的结构，得  $\mathbf{F}$  的结构为  $Cl_3CNCCl_2$ 。

# CHECKPOINT

1 PTS

$$
\mathbf {F} = C l _ {3} C N C C l _ {2}
$$

$\mathbf{F}$  的结构与  $\mathbf{C}$  的结构相似, 可以视为  $\mathbf{C}$  的羟基替换为三氯甲基, 因此说法6正确。

$\mathbf{H}$  与  $\mathbf{F}$  相对分子质量相差16，则  $\mathbf{H} = Cl_3CN(O)CCl_2$

# CHECKPOINT

0.5 PTS

$$
\mathbf {H} = C l _ {3} C N (O) C C l _ {2}
$$

E 在  $120^{\circ} \mathrm{C}$  下反应方程式为  $C l_{3} C N O = C C l_{4} + C l_{3} C N (O) C C l_{2} + N O$ , 符合产物比  $1: 1: 2$

H中的  $N - O$  键存在电荷分离，说法5错误。

# CHECKPOINT

0.5 PTS

H中存在电荷分离，说法5错

综上，1和5错误，选J。
