# 答案

正确答案: C

# 详细解析

草酸转移电子的总量为：

$$
2 \times 2 5. 0 0 \mathrm {m L} \times 0. 1 0 1 5 \mathrm {m o l} / \mathrm {L} = 5. 0 7 5 \times 1 0 ^ {- 3} \mathrm {m o l}
$$

草酸转移电子的总量为  $5.075 \times 10^{-3} \mathrm{~mol}$

高锰酸钾转移电子的总量为:

$$
0. 0 2 0 3 0 \mathrm {m o l} / \mathrm {L} \times 2 1. 8 4 \mathrm {m L} \times 5 = 2. 2 1 7 \times 1 0 ^ {- 3} \mathrm {m o l}
$$

高锰酸钾转移电子的总量为:  $2.217 \times 10^{-3} \mathrm{~mol}$

所以  $1.000\mathrm{g}$  X对应的电子数为：

$$
5. 0 7 5 \times 1 0 ^ {- 3} \mathrm {m o l} - 2. 2 1 7 \times 1 0 ^ {- 3} \mathrm {m o l} = 2. 8 5 8 \times 1 0 ^ {- 3} \mathrm {m o l}
$$

设  $\mathbf{X}$  的化学式为  $\mathrm{Ag_xPbO_y}$ ，并假设其中  $\mathrm{Ag}$  和  $\mathrm{O}$  分别为  $+1$  价、-2价，则：

$$
\begin{array}{l} (2 0 7. 2 + 3 2. 0 7 + 1 6. 0 0 \times 4) / (1 0 7. 9 x + 2 0 7. 2 + 1 6. 0 0 y) = 0. 2 8 6 / 0. 5 0 0 (a) \\ 1. 0 0 0 \mathrm {g} \times (2 \mathrm {y} - \mathrm {x} - 2) / (1 0 7. 9 \mathrm {x} + 2 0 7. 2 + 1 6. 0 0 \mathrm {y}) = 2. 8 5 8 \times 1 0 ^ {- 3} \mathrm {m o l} (\mathrm {b}) \\ \end{array}
$$

列出两个方程组，分别为  $(207.2 + 32.07 + 16.00 \times 4) / (107.9\mathrm{x} + 207.2 + 16.00\mathrm{y}) = 0.286 / 0.500$  和  $1.000\mathrm{g} \times (2\mathrm{y} - \mathrm{x} - 2) / (107.9\mathrm{x} + 207.2 + 16.00\mathrm{y}) = 2.858 \times 10^{-3}\mathrm{mol}$
