# 题目

石墨炔是一种新型的碳材料。与石墨烯类似，石墨炔通过  $\mathsf{sp}^2$  和sp杂化的碳构成单层的网络结构层与层之间通过范德华力连接。因  $\mathsf{sp}^2$  杂化的碳和sp杂化的碳可以有多种排列方式，所以石墨炔可以有多种结构。其中一种石墨炔，每个  $C(sp^2)$  与三个  $C(sp)$  连接，每个  $C(sp)$  分别与一个  $C(sp)$  和一个 $C(sp^2)$  连接，我们在此暂时命名为石墨炔A；另一种石墨炔，每个  $C(sp^2)$  分别与两个  $C(sp)$  和一个 $C(sp^2)$  连接，每个  $C(sp)$  分别与一个  $C(sp)$  和一个  $C(sp^2)$  连接，我们在此暂时命名为石墨炔B；此外还有一种石墨炔结构是由  $[C(sp^2)C(sp)C(sp)C(sp^2)]$  组成的网络结构，但含有苯环的结构，我们在此暂时命名为石墨炔  $C$  。

石墨炔  $C$  的变体石墨双炔可以以六炔苯为单体进行在任意基底上的合成。方法为将基底材料包裹在某金属网中，浸入含有六炔苯的吡啶和四甲基乙二胺的丙酮溶液，即可以在基底上发生Glaser-Hay炔偶联反应，得到生长的石墨双炔。

结合以上背景资料，以下说法错误的是（）。

A. 石墨炔  $A$  正当晶胞中有六个sp杂化的碳  
B. 金属网元素对应离子稳定价态有两种。  
C. 石墨炔  $C$  可以由含三个溴原子的单体合成。  
D. 石墨炔  $A$  可以由含两个溴原子的单体合成, 这个单体不可能有顺反异构体。  
E. 石墨炔  $A$  与石墨炔  $C$  正当晶胞所含碳原子数比值为  $2: 3$  。  
F. 石墨炔  $C$  采用ABAB层状堆叠（共中心大小环交错），钠离子嵌入夹层后形成NaC组成（即  $\mathrm{Na} : \mathrm{C} = 1: 1$ ）。若其俯视图中多边形代表碳骨架孔洞（不画碳原子），圆点代表钠离子，则钠离子位于A层与B层投影重叠形成的正六边形中心，每六边形一个钠离子。

# 答案

正确答案: F

# 详细解析

A: 石墨炔  $A$  是由  $\left[C \left(s p^{2}\right) C \left(s p\right) C \left(s p\right) C \left(s p^{2}\right)\right]$  循环组成的六边形的 18 元环排列的, 根据正当晶胞定义与划分原则, 其正当晶胞是一个包含 8 个碳原子的菱形[相邻两行的四个 18 元环 (上下两行各取两个 18 元环) 中心相连组成的菱形], 其中含有 6 个 sp 杂化的碳和 2 个 sp2 杂化的碳。

# CHECKPOINT

2 PTS

石墨炔  $A$  的正当晶胞是一个包含 8 个碳原子的菱形[相邻两行的四个 18 元环（上下两行各取两个 18 元环）中心相连组成的菱形]

B：铜催化的末端炔烃的交叉偶联反应(Glaser-Hay reaction)是有机化学的一个基本反应，推断金属网为铜网， $\mathrm{Cu}(\mathrm{py})2+$  可迁移至基底上进行吸附，进而催化偶联反应发生在基底表面，铜离子稳定价态包括+1和+2。

# CHECKPOINT

2 PTS

铜催化的末端炔烃的交叉偶联反应(Glaser-Hay reaction)

C: 石墨炔  $C$  可以由单体  $\mathrm{C} \# \mathrm{CC} 1 = \mathrm{C}(\mathrm{Br}) \mathrm{C}(\mathrm{C} \# \mathrm{C}) = \mathrm{C}(\mathrm{Br}) \mathrm{C}(\mathrm{C} \# \mathrm{C}) = \mathrm{C} 1 \mathrm{Br}$  合成, 其含有 3 个溴原子。

# CHECKPOINT

1 PTS

石墨炔  $C$  可以由单体  $\mathrm{C}\# \mathrm{CC}1 = \mathrm{C}(\mathrm{Br})\mathrm{C}(\mathrm{C}\# \mathrm{C}) = \mathrm{C}(\mathrm{Br})\mathrm{C}(\mathrm{C}\# \mathrm{C}) = \mathrm{C}1\mathrm{Br}$  合成

D: 石墨炔  $A$  可以由  $\mathrm{Br} / \mathrm{C}(\mathrm{Br}) = \mathrm{C}(\mathrm{C}\# \mathrm{C})\backslash \mathrm{C}\# \mathrm{C}$  单体合成, 并且没有顺反异构。

# CHECKPOINT

1 PTS

石墨炔  $A$  可以由  $\mathrm{Br} / \mathrm{C}(\mathrm{Br}) = \mathrm{C}(\mathrm{C}\# \mathrm{C})\backslash \mathrm{C}\# \mathrm{C}$  单体合成, 并且没有顺反异构

E.石墨炔正  $C$  当晶胞中包含一个苯环与六个sp杂化的碳，共12个碳原子。如关于A选项的解析所述，石墨炔A正当晶胞中含有六个sp杂化的碳和两个sp2杂化的碳，共8个碳。

# CHECKPOINT

1 PTS

石墨炔  $C$  正当晶胞中包含一个苯环与六个sp杂化的碳。石墨炔  $A$  正当晶胞中含有六个sp杂化的碳和两个sp2杂化的碳。

F.ABAB堆叠意味着相邻两层（A和B）的环不是对齐的，而是错位的共中心大小环交错，因此俯视图中A层和B层的多边形投影应错开。题目描述"钠离子填在夹层里且每层钠离子位置相同"，说明钠离子位于两层之间。若钠离子位于重叠区域中心，AB层错位后重叠区域可能不居中。“钠离子位于A层与B层投影重叠形成的正六边形中心”是错误的，因为AB堆叠的错位导致投影重叠区域不再是原六边形的中心。故层间钠离子应占据相邻三个多边形的交点投影，类似石墨中锂离子的嵌入位点。钠离子位于相邻A、B层多边形顶点交汇的三叉空隙，每个三叉点一个钠离子。

# CHECKPOINT

2 PTS

若钠离子位于重叠区域中心，AB层错位后重叠区域可能不居中。
