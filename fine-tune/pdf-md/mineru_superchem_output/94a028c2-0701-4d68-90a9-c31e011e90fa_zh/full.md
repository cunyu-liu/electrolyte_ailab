# 题目

现在有如下转化过程：

![](images/56a7ef7a13c42158d69a6bc48587ce83fec025ac73e19a7d7f5aea3e8cb037a3.jpg)

化合物{SMILES编码为C[C@]12CCC(=O)C=C2CCC1=O}通过NaBH4处理后得到化合物A。化合物A在H3PO4催化下和化合物{SMILES编码为C=C(C)C}反应，得到化合物B。化合物B在tBuOK催化下和过量CH3I反应得到化合物C。化合物C在Pt/C催化下与H2反应得到化合物D。化合物D在HCl处理后得到化合物E。化合物E在NaOAc存在条件下与PCC试剂反应得到化合物F。化合物F在p-TsOH催化下和化合物{SMILES编码为OCCO}反应得到化合物G。化合物G在NaOAc存在条件下和NH2OH·HCl在EtOH中反应得到化合物H。

请分析各个化合物结构，并选出正确选项。

A. 化合物A为

![](images/4d9ffe9dd3e144486ffa4f7be257b6313818431215a7ee7dfd83bd44d6a47b44.jpg)

C[C@@]12CCC(C=C1CCC2=O)O

# B. 化合物B为

![](images/c39f3a51fb62b93a30117392d4f721898becf6cd90c73c727b90d91209d790b9.jpg)

C[C@@]12CCC(C=C1CCC2OCC(C)C)=O

# C. 化合物C为

![](images/7ea35d8d21e81e72401df75a12953a14f415b9f349201983de51af68b6fd1c56.jpg)

C[C@@]12CC=C(C=C1CCC2OC(C)(C)C)OC

# D. 化合物D为

![](images/3ff3e5615ff3cf02f2f262d8e94758f9b2e8c07fa15c878dea47027d527a393e.jpg)

C[C@@]12CC=C(CC1CCC2OCC(C)C)OC

# E. 化合物E为

![](images/d6871f3c6a16c64a7e05e89e09f36ba523147b50d664447f2c09e9fa4062609d.jpg)

C[C@@]12CCC(CC1CCC2O)=O

# F. 化合物F为

![](images/fba5f9e48509dad06e54a091bb86a6974d930f010de57021d1c618470ad9fe5b.jpg)

C[C@@]12CCC(C(C)(C)C1CCC2=O)=O

# G. 化合物G为

![](images/3474e63534122f866416c16b73098fb04d96eb417e04400269d595cd37c12071.jpg)

CC(C1CCC2=O)(C3(CC[C@]21C)OCCCO3)C

# H. 化合物H为

![](images/d31c9961a747c3758df0e9a8078b9a62cc76b85750141369e43407f12c0dc10b.jpg)

C[C@@]12CCC3(OCC03)C(C)C1CC/C2=N\O

# 答案

正确答案: F

# 详细解析

硼氢化钠先还原不共轭羰基，且体系中后续仍有酮α位酸性氢，故而此时应只还原不共轭羰基，化合物A为

![](images/c9c8f6f8ce256839cf54ff777662269398708c0d8078d6c1a7621319462ab818.jpg)

C[C@@]12CCC(C=C1CCC2O)=O

CHECKPOINT

1 PTS

化合物A为C[C@@]12CCC(C=C1CCC2O)=0

酸性条件下，2-甲基丙烯生成叔丁基正离子，进而与羟基结合，故化合物B为

![](images/e81c0048b9f0d1fb3b1fff3e933fe68785fee5f2ba93ba27985d464a07fd182f.jpg)

C[C@@]12CCC(C=C1CCC2OC(C)(C)C)=O

# CHECKPOINT

1 PTS

化合物B为C[C@@]12CCC(C=C1CCC2OC(C)(C)C)=O

接着碱性下拔出线性共轭的最强酸性的氢，对过量碘甲烷亲核进攻两次，进而化合物C为

![](images/6d872b9480168a1049e1fc2a77ddf171dd06b1cd3f0bd16ffb87011701c28646.jpg)

C[C@@]12CCCC(C(C)(C)C1=CCC2OC(C)(C)C)=O

# CHECKPOINT

1 PTS

化合物C为C[C@@]12CCC(C(C)(C)C1=CCC2OC(C)(C)C)=O

而钯碳条件下氢气能还原碳碳双键，进而化合物D为

![](images/1442e12aa8262a25a4e440a083f47041ee8876e7f8bafc551f2d2d11e4507748.jpg)

C[C@@]12CCC(C(C)(C)C1CCC2OC(C)(C)C)=O

# CHECKPOINT

1 PTS

化合物D为C[C@@]12CCC(C(C)(C)C1CCC2OC(C)(C)C)=O

酸处理下，叔丁基保护基会去保护，进而化合物E为

![](images/440826cc9e1144f5af979695153f7b83e6c7fa5c55287eb46bf58d82fd50d610.jpg)

C[C@@]12CCC(C(C)(C)C1CCC2O)=O

# CHECKPOINT

1 PTS

化合物E为C[C@@]12CCC(C(C)(C)C1CCC2O)=0

吡啶氯铬酸盐可以将羟基氧化为羰基，故而化合物F为

![](images/15315202918bd4702792db1fe834b8d9809f694df9ee95b7705400a71b8136fa.jpg)

C[C@@]12CCC(C(C)(C)C1CCC2=O)=O

# CHECKPOINT

1 PTS

化合物F为C[C@@]12CCC(C(C)(C)C1CCC2=O)=O

而酸性下乙二醇能保护羰基，且后续还进一步发生了涉及羰基的反应。进而此时仅保护了一个羰基，而六元环上羰基会优先保护，进而化合物G为

![](images/71388483dd91ba6c7607eed6a3c4ca47b4b31c293e9af704acfac6ddf8316efd.jpg)

C[C@@]12CCC3(OCC03)C(C)(C)C1CCC2=O

# CHECKPOINT

2 PTS

化合物G为C[C@@]12CCC3(OCC03)C(C)(C)C1CCC2=O

最后羟胺能和羰基缩合，并考虑位阻，故而化合物H为

![](images/0600796c4fda869eff9be2d8904d962065df8a58aa025c85cd2a7962587145a1.jpg)

C[C@@]12CCC3(OCCO3)C(C)(C)C1CC/C2=N\0

# CHECKPOINT

2 PTS

化合物H为C[C@@]12CCC3(OCC03)C(C)(C)C1CC/C2=N\O

从而选项F正确。
