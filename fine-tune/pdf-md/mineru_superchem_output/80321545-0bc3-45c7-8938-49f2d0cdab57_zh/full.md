# 题目

某种短周期元素  $\mathbf{X}$  常应用于核动力反应堆中。其某种卤化物在  $\mathrm{N}_{2} \mathrm{O}_{4}$  中充分反应, 得到一种白色晶体  $\mathrm{A}$  。

现在对A进行热重分析,已知其有两个失重峰,首先失重  $58.04\%$  得到白色物质B,继续加热,再失重 $8.52\%$  得到一种具有特殊结构的物质C。C中含有4个  $\mathrm{C}_{3}$  轴,所有X化学环境相同,且配位数均为4。

无独有偶,人们在很多过渡金属中也发现了具有类似C的结构的物质。对于第四周期的某金属组成的类似粒子M中,配体含有乙酸根,该金属的质量分数为  $26.88\%$ , 所有金属原子化学环境相同,且均为6配位。已知该粒子M可以在水溶液中制备。

下列说法正确的是：

A. 其他选项均不正确  
B. X原子基态含有两个单电子  
C. A 的原子数大于 C  
D. M中氧的质量分数为  $39.5\%$  
E. M中氧的质量分数为  $44.1\%$  
F. M 的对称性大于 C  
G. M为中性粒子

# 答案

正确答案: E

# 详细解析

短周期元素X常应用于核动力反应堆中，在核反应堆中，Be常用作中子慢化剂和反射剂，B常用作中子吸收剂。因此X可能为B,Be。

1 PTS

X可能为B，Be

X 的卤化物与  $\mathrm{N}_{2} \mathrm{O}_{4}$  反应, 一般得到无水硝酸盐与  $\mathrm{N}_{2} \mathrm{O}_{4}$  的加合物; 考虑第一步失重分子量很大, 很有可能失去加和的  $\mathrm{N}_{2} \mathrm{O}_{4}$  。

# CHECKPOINT

1 PTS

卤化物与  $\mathrm{N}_{2} \mathrm{O}_{4}$  反应, 一般得到无水硝酸盐与  $\mathrm{N}_{2} \mathrm{O}_{4}$  的加合物

# CHECKPOINT

1 PTS

第一步失重失去加和的  $\mathrm{N}_{2} \mathrm{O}_{4}$

若  $\mathbf{X}$  为B，A可写为  $\mathrm{B(NO_3)_3\cdot xN_2O_4}$  。首先B的硝酸盐并不常见；其次x无论取任何值都无法对应该失重峰。因此X不为B。

# CHECKPOINT

1 PTS

根据失重比例判断X不为B

若  $\mathbf{X}$  为  $\mathrm{Be}$ ， $\mathbf{A}$  可写为  $\mathrm{Be}(\mathrm{NO}_3)_2 \cdot \mathrm{xN}_2\mathrm{O}_4$ 。此时当  $\mathrm{x}$  取2时，第一步失重比例为  $184 / 317 = 58.0\%$ ，符合题设。

因此x为2，A化学式为  $\mathrm{Be(NO_3)_2\cdot 2N_2O_4}$  。X为Be，选项B错误。

# CHECKPOINT

1 PTS

A 化学式为  $\mathrm{Be}(\mathrm{NO}_3)_2 \cdot 2\mathrm{N}_2\mathrm{O}_4$

C中含有4个  $\mathrm{C}_{3}$  轴,所有X化学环境相同,且配位数均为4。首先判断该化合物为四面体结构;因此大概率含有四个Be。很容易想到Be化合物的经典结构  $\mathrm{Be}_{4} \mathrm{O}(\mathrm{NO}_{3})_{6}$ , 根据失重峰可判断该化学式正确。因此C为  $\mathrm{Be}_{4} \mathrm{O}(\mathrm{NO}_{3})_{6}$  。根据化学式可知选项C错误。

# CHECKPOINT

1 PTS

C为四面体结构；含有四个Be

# CHECKPOINT

1 PTS

C 为  $\mathrm{Be}_4\mathrm{O}(\mathrm{NO}_3)_6$

M 形成的粒子根据题设，应当具有类似的金属-氧簇结构，且有乙酸根作为配体。化学式可尝试写为  $\mathrm{M_xO(CH_3COO)_y}$ ；

# CHECKPOINT

1 PTS

M形成的粒子具有类似的金属-氧簇结构

考虑M的质量分数，由于M为六配位，在氧簇提供1个配位数的情况下，配体需要提供5配位数；若该粒子与  $\mathrm{Be}_4\mathrm{O}(\mathrm{NO}_3)_6$  结构一致为四面体结构，乙酸根作为桥连配体的情况下，M的配位数最多为4，所以存在其他配体提供配位数。

考虑该粒子于水中制备，考虑该配体为水；

# CHECKPOINT

1 PTS

存在其他配体提供配位数

# CHECKPOINT

1 PTS

存在配体为水

化学式可尝试写为  $\mathrm{M_xO(CH_3COO)_y(H_2O)_z}$  ；考虑x取4，此时乙酸根作为桥连配体位于四面体的边上，即 $y = 6$  ；为使得M的配位数为6，每个M需要额外与两个水配位。因此z取8;

根据质量分数  $26.88\%$  计算  $\mathbf{M}$  的相对原子质量为47.2，无合理金属；因此该粒子不含四个  $\mathbf{M}$  。

# CHECKPOINT

1 PTS

通过质量分数推断，该粒子不含四个M

考虑对称性下降，粒子含有三个M，此时乙酸根作为桥连配体，平面三角形边可以存在两个乙酸根，因此y可以取6或3，对应满足配位数的z可以取3或9。

根据质量分数计算  $\mathbf{M}$  的相对原子质量：当  $y = 6, z = 3$  时，原子量为52.08，为  $\mathrm{Cr}$ ；当  $y = 3, z = 9$  时为43.5，无合理金属。

因此该粒子的化学式为  $\mathrm{Cr_3O(CH_3COO)_6(H_2O)_3}$ ，考虑  $\mathrm{Cr}$  六配位的常见氧化态  $+3$ ，该粒子应当带  $+1$  电荷。因此选项F,G均错误。
