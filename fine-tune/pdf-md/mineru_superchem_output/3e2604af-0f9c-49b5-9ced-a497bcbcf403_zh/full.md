# 题目

硫化铵是广泛使用的定性分析化学试剂，配制此试剂时，将硫化氢气体导入  $4 \sim 5 \mathrm{~mol} / \mathrm{L}$  的氨水中，之后再加水。用这个方法无法配制得到纯的溶液：当吸收了过低或过高于化学计量的气体时，溶液中会含有过量的氨或硫氢化铵。

将  $10.00 \mathrm{~mL}$  的硫化铵试液稀释到  $1.000 \mathrm{~L}$ , 取  $10.00 \mathrm{~mL}$  所得到的储存溶液移至一个蒸馏瓶并且加入大约  $40 \mathrm{~mL}$  的水。然后, 将  $25.00 \mathrm{~mL}$  的  $0.1 \mathrm{~mol} / \mathrm{L}$  硝酸镉溶液加到收集瓶(蒸馏出的成分将凝结于此)。并且将  $20.00 \mathrm{~mL}$  的  $0.02498 \mathrm{~mol} / \mathrm{L}$  硫酸溶液加入蒸馏瓶。

将蒸馏瓶中大约一半的溶液蒸馏至收集瓶。(收集瓶中可以看到黄色的沉淀物。)

将留在蒸馏瓶中的物质完全洗涤至锥形瓶，加入数滴甲基红指示剂之后以  $0.05002 \mathrm{~mol} / \mathrm{L}$  NaOH溶液滴定，终点时消耗NaOH溶液  $10.97 \mathrm{~mL}$  。

# 答案

正确答案: I

# 详细解析

留在蒸馏瓶中的离子有  $\mathrm{NH}_{4}^{+}$ 和  $\mathrm{H}^{+}$ , 以甲基红为指示剂只会消耗掉  $\mathrm{H}^{+}$

剩下的  $\mathrm{NH_4^+}$  都是  $\mathrm{H}_2\mathrm{SO}_4$  质子化  $\mathrm{NH_4^+}$  产生的，因此：

$$
\mathrm {c} \left(\mathrm {N H} _ {3}\right) = \left(0. 0 2 0 0 0 \times 0. 0 2 4 9 8 \times 2 - 0. 0 5 0 0 2 \times 0. 0 1 0 9 7\right) \times \frac {1 0 0}{0 . 0 1 0 0 0} = 4. 5 0 5 (\mathrm {M})
$$

硫化铵试剂溶液含  $4.505 \mathrm{M}$  的  $\mathrm{NH}_{3}$

接收瓶发生了两个反应：

$$
\mathrm {C d} ^ {2} + + \mathrm {H} _ {2} \mathrm {S} = \mathrm {C d S} + 2 \mathrm {H} ^ {+}

$$
\mathrm {C d S} + 4 \mathrm {B r} _ {2} + 4 \mathrm {H} _ {2} \mathrm {O} = \mathrm {C d S O} _ {4} + 8 \mathrm {H B r}
$$

因此一当量  $\mathrm{H}_2\mathrm{S}$  产生十当量  $\mathrm{H}^+$  。
