# 题目

现有以下化合物，可被用作高分子链式聚合的单体：

乙烯，丙烯，异丁烯，苯乙烯，丁二烯，丙烯酸甲酯，氯乙烯，醋酸乙烯酯，甲基乙烯基醚，丙烯酸，丙烯腈

对于以上物质，有以下命题：

1. 以上单体均可进行自由基链式聚合形成高分子  
2. 以上单体中, 至少有四种可以进行正离子聚合形成高分子, 其中有一种难以形成高分子量聚合物  
3. 以上单体中，可以进行负离子聚合的物种有四种。  
4. 以上单体中，含氮原子取代基的单体数量，与可以进行所有四种主要类型聚合（自由基、阳离子、负离子、配位）的单体数量相等  
5. 以上单体中，不含芳香环的几种烯烃单体，它们都能进行同一种聚合反应

经过分析判断以上命题中正确的命题数量。

A. 0  
B. 1  
C. 2  
D. 3  
E. 4  
F. 5

# 答案

正确答案: D

# 详细解析

逐一分析每个命题：

命题1：以上单体均可进行自由基链式聚合形成高分子

反例：异丁烯，其双键碳上连接两个甲基，空间位阻很大，且极易发生链转移反应，难以形成高分子。它主要通过阳离子聚合。故该命题错误。

# CHECKPOINT

1 PTS

异丁烯空间位阻很大，主要通过阳离子聚合

# CHECKPOINT

0.5 PTS

命题1错误，可举出反例异丁烯

## 命题 2: 以上单体中, 至少有四种可以进行正离子聚合, 其中有一种难以形成高分子量聚合物

能够进行阳离子聚合的单体需要其取代基能稳定碳正离子，如异丁烯（生成稳定的叔碳正离子）、苯乙烯（生成共轭稳定的苄基正离子）、丁二烯（生成共轭稳定的烯丙基正离子）、甲基乙烯基醚（氧原子p-π共轭效应稳定碳正离子）。

# CHECKPOINT

1 PTS

能够进行阳离子聚合的单体需要其取代基能稳定碳正离子

# CHECKPOINT

1 PTS

异丁烯、苯乙烯、丁二烯、甲基乙烯基醚均有取代基稳定正离子的能力，可进行正离子聚合

丙烯虽然其甲基有给电子效应，但生成的仲碳正离子不够稳定，极易发生链转移重排，导致链终止，因此难以形成高分子量聚合物。

# CHECKPOINT

1 PTS

丙烯生成的仲碳正离子不够稳定，难以形成高分子量聚合物，命题2正确。

命题3：以上单体中，有四种可以进行负离子聚合的物种。

可进行负离子聚合的单体需要强吸电子基或共轭体系。例如苯乙烯、丁二烯、丙烯酸甲酯、丙烯腈。

# CHECKPOINT

可进行负离子聚合的单体需要强吸电子基或共轭体系

# CHECKPOINT

1 PTS

苯乙烯、丁二烯、丙烯酸甲酯、丙烯腈可进行负离子聚合

数量为4，因此命题3正确。

# CHECKPOINT

0.5 PTS

可以进行负离子聚合的物种为4，因此命题3正确。
