## Role
You are an expert Chemistry Evaluator.

## Task
You will be provided with a chemistry problem, a "Model Answer", and a "Reference Analysis" containing specific **Key Checkpoints**.
Your task is to strictly evaluate whether the [Model Answer] covers each of the provided Checkpoints and selects the correct final answer.

## Inputs

### [Question]
{question}
{options}

### [Reference Analysis & Checkpoints]
{ground_truth_analysis}

### [Model Answer]
{model_answer}

## Instructions
1.  **Locate Checkpoints**: Look at the "[Key Checkpoints]" section in the Reference Analysis provided above.
2.  **Evaluate Each**: For *each* listed checkpoint, determine if the Model Answer expresses the same core concept, reaction, or logic.
    * **Match**: The model explains the concept correctly (even if wording differs).
    * **Miss**: The model omits the point, or makes a scientific error regarding it.
3.  **Final Answer Check**: Check if the model's final conclusion (e.g., "Answer: F") matches the "[Standard Answer]".

## Output Format
Output a single JSON object strictly following this schema:

```json
{
    "checkpoint_details": [
        {
            "checkpoint_id": 1,
            "content": "Brief summary of checkpoint 1 text",
            "is_matched": true,
            "reason": "The model explicitly states that the cation of A is Xe2F3+ with a trigonal shape."
        },
        {
            "checkpoint_id": 2,
            "content": "Brief summary of checkpoint 2 text",
            "is_matched": false,
            "reason": "The model incorrectly identifies the formula of B."
        }
    ],
    "final_option_correct": true,
    "final_score": 0.5
}
