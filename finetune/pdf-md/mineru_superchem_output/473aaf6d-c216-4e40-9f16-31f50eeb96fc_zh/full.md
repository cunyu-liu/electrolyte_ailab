# 答案

正确答案: E

# 详细解析

由  $\mathrm{pM} = \rho \mathrm{RT}$  可知，第三步失去的气体分子量为44.0，为  $\mathrm{CO}_{2}$

# CHECKPOINT

1 PTS

第三步失去的气体为  $\mathrm{CO}_{2}$

A 为碱式碳酸盐，前两步只能是失水。假设第一步失去结晶水，第二步失去羟基水。

结晶水摩尔数  $\mathrm{n}_{1}(\mathrm{H}_{2}\mathrm{O}) = 2.524\mathrm{g}\times 12.19\% /18.02\mathrm{g}\cdot \mathrm{mol}^{-1} = 0.01707\mathrm{mol}$

# CHECKPOINT

0.5 PTS

结晶水为  $0.01707 \mathrm{~mol}$

羟基水摩尔数  $\mathrm{n}_{2}(\mathrm{H}_{2}\mathrm{O}) = 2.524\mathrm{g}\times 9.26\% /18.02\mathrm{g}\cdot \mathrm{mol}^{-1} = 0.01297\mathrm{mol}$

# CHECKPOINT

0.5 PTS

羟基水为  $0.01297 \mathrm{~mol}$

$$
\mathrm {n} \left(\mathrm {CO} _ {2}\right) = [ 2.524 \mathrm {g} \times (1 - 12.19 \% - 9.26 \%) - 1.603 \mathrm {g} ] / 44.01 \mathrm {g} \cdot \mathrm {mol} ^ {- 1} = 0.008625 \mathrm {mol}
$$

# CHECKPOINT

0.5 PTS

第三步放出  $0.008625 \mathrm{~mol} \mathrm{CO}_{2}$

假设剩下的固体中  $\mathbf{M}$  为  $+2$  价，则MO摩尔数为（对应  $\mathrm{CO_2}$  和羟基水）： $\mathrm{n(MO)} = 0.01297\mathrm{mol} + 0.008625\mathrm{mol} = 0.02160\mathrm{mol}$

# CHECKPOINT

0.5 PTS
