# 题目

二氧化碳气敏电极由透气膜、  $0.010\mathrm{mol}\cdot \mathrm{L}^{-1}$  的  $\mathrm{NaHCO_3}$  溶液、  $\mathrm{pH}$  电极和外参比电极组成。外参比电极为标准甘汞电极，  $\varphi (\mathrm{Cl}^{-}|\mathrm{Hg}_{2}\mathrm{Cl}_{2}|\mathrm{Hg}) = 0.2801\mathrm{V}$  。计算大气中二氧化碳质量分数为611ppmbymass时原电池的电极电势。已知：  $\mathrm{CO}_{2}$  的亨利常数  $K_{\mathrm{H}} = 3.0\times 10^{3}\mathrm{kPa}\cdot \mathrm{L}\cdot \mathrm{mol}^{-1}$  ， $\mathrm{CO}_{2} + \mathrm{H}_{2}\mathrm{O} = \mathrm{H}_{2}\mathrm{CO}_{3}$  的  $K = 1.70\times 10^{-3}$  碳酸的电离常数  $\mathrm{pK_{a1,2}} = 6.35,10.33$  。

A.  $0.254 \mathrm{~V}$  
B.  $0.397 \mathrm{~V}$  
C.  $0.543 \mathrm{~V}$  
D.  $0.689 \mathrm{~V}$  
E.  $0.731 \mathrm{~V}$  
F.  $0.826 \mathrm{~V}$  
G.  $1.031 \mathrm{~V}$  
H.  $1.102 \mathrm{~V}$

# 答案

正确答案: F

# 详细解析

由  $\omega_{\mathrm{CO_2}} = 611~\mathrm{ppm}$  by mass，即  $\omega_{\mathrm{CO_2}} = 6.11\times 10^{-4}$  ，以及摩尔质量和标准大气压数据 $M_{\mathrm{CO_2}} = 44\mathrm{g}\cdot \mathrm{mol}^{-1},\bar{M}_{\mathrm{air}} = 29\mathrm{g}\cdot \mathrm{mol}^{-1},1\mathrm{atm} = 101.325\mathrm{kPa},$  计算得到： $p_{\mathrm{CO_2}} = \omega_{\mathrm{CO_2}}\times \frac{\bar{M}_{\mathrm{air}}}{M_{\mathrm{CO_2}}}\times 1\mathrm{atm} = 0.0403\mathrm{kPa}.$

# CHECKPOINT

$$
p _ {\mathrm {C O} _ {2}} = 0. 0 4 0 3 \mathrm {k P a}
$$

由亨利定律，溶液中溶解平衡时的  $\mathrm{CO}_{2}$  的浓度为  $[\mathrm{CO}_{2}] = \frac{p\mathrm{CO}_{2}}{K_{\mathrm{H}}} = 1.36\times 10^{-5}\mathrm{mol}\cdot \mathrm{L}^{-1}$  。

# CHECKPOINT

在该体系中， $\left[\mathrm{HCO}_{3}^{-}\right]$  远大于  $\left[\mathrm{H}_{2} \mathrm{CO}_{3}\right]$ ，故可近似认为  $\left[\mathrm{HCO}_{3}^{-}\right] = 0.010 \mathrm{~mol} \cdot \mathrm{L}^{-1}$  不变。

# CHECKPOINT

$\left[\mathrm{HCO}_{3}^{-}\right] = 0.010 \mathrm{~mol} \cdot \mathrm{L}^{-1}$  不变
