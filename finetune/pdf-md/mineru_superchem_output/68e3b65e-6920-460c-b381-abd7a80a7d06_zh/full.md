# 题目

在一特制的密封体系中,  $\left[\mathrm{n}-\mathrm{C}_{4} \mathrm{H}_{9}\right]_{4} \mathrm{~N}\left[\mathrm{H}_{2} \mathrm{PO}_{4}\right]$  与  $\mathrm{HSiCl}_{3}$  反应6天, 可得到无色针状化合物A并放出气体, 产物共有三种（反应1）。进一步分析表明, A是一种1:1型盐, 其摩尔质量为  $542.3 \mathrm{~g} / \mathrm{mol}$  。在A的阴离子中仅含有P, Si, Cl三种元素; 同种原子具有相同的化学环境。

A对水敏感，1mol A遇水分解生成1mol剧毒气体（反应2）。

在  $382 \mathrm{~K}$  下  $\mathbf{A}$  与  $\mathrm{Ph}(\mathrm{CH}_{2})_{4} \mathrm{Cl}$  充分反应后加水处理，得到二取代膦（反应3）。

下列说法正确的是：

A. 其他选项均不正确  
B. A 的阴离子含有正四面体结构  
C. 反应1的化学方程式，以最简系数比配平后，产物的系数与反应物系数和相等。  
D. 反应2的化学方程式, 以最简系数比配平后, 不含 Si 的产物的系数和为合数。  
E. 反应3的二取代膦产物含有小于40个原子

# 答案

正确答案: C

# 详细解析

A的阳离子很明显为  $\left[\left(\mathrm{n}-\mathrm{C}_{4} \mathrm{H}_{9}\right)_{4} \mathrm{~N}\right]^{+}$ ; 由于为  $1:1$  型盐, 阴离子的化学式可设为  $\mathrm{P}_{\mathrm{x}} \mathrm{Si}_{\mathrm{y}} \mathrm{Cl}_{\mathrm{z}}$  。

# CHECKPOINT

1 PTS

A的阳离子为  $\left[\left(\mathrm{n}-\mathrm{C}_{4} \mathrm{H}_{9}\right)_{4} \mathrm{~N}\right]^{+}$

阴离子的相对分子质量为  $542.3 - (14 + (12 \times 4 + 9) \times 4) = 300.3$  。考虑同种原子具有相同的化学环境，z很可能为偶数。先假设  $x = y = 1$  ，此时  $z = (300.3 - 31 - 28.1) / 35.45 = 6.8$  ，不为整数。

设  $x = 1, y = 2$  ， $z = (300.3 - 31 - 28.1 \times 2) / 35.45 = 6$  ，为整数。观察化合价， $\mathrm{Cl}$  很明显为-1，此时  $\mathrm{Si}$  为+4， $\mathrm{P}$  为-3，均为合理化合价。因此  $\mathbf{A}$  的阴离子为  $\mathrm{PSi}_2\mathrm{Cl}_6^-$ ，化学式为  $[(n - C_4H_9)_4N][\mathrm{PSi}_2\mathrm{Cl}_6]$ 。
