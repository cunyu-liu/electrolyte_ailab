# 题目

-  $78^{\circ} \mathrm{C}$  下,  $\mathrm{Cl}_{2} 、 \mathrm{NCI}_{3}$  和  $\mathrm{AsF}_{5}$  在液态  $\mathrm{SO}_{2}$  溶剂中反应产生一种盐  $\mathbf{A}$ ,  $\mathbf{A}$  的阴阳离子都具有高的对称性, 参与反应的  $\mathrm{Cl}_{2}$  和  $\mathrm{NCI}_{3}$  的摩尔比为 1:2。升温时  $\mathbf{A}$  发生分解, 产物中没有单质。将  $\mathbf{A}$  与  $\mathrm{CF}_{3} \mathrm{SCI}$  反应产生盐  $\mathbf{B}$  和几种小分子的二元化合物, 原有  $\mathrm{S}-\mathrm{Cl}$  键未被破坏,  $\mathbf{B}$  中阳离子对称性与  $\mathrm{NH}_{3}$  相同。

对于上述过程，下列说法中错误的是

A. A和B的阴离子相同，且均为  $\mathrm{O_h}$  点群  
B. A 的阳离子为  $\mathrm{T}_{\mathrm{d}}$  点群  
C. 生成  $\mathbf{A}$  的过程中, 配平后化学方程式系数的最简整数比的乘积为  $12$  
D. A 分解过程中产生的气体在潮湿状态会腐蚀玻璃容器, 因此需要保证实验严格无水  
E. 生成  $\mathbf{B}$  过程中,  $\mathbf{B}$  中阳离子和产物中的一种互为等电子体  
生成B的过程中，配平后化学方程式系数的最简整数比的和为8  
G. 以上说法均正确  
H. 其他说法均不正确

# 答案

正确答案: G

# 详细解析

这是一道无机物推断。题干中指出，“A的阴阳离子都具有高的对称性”，考虑反应物中能参与阴离子形成的只有  $\mathrm{AsF}_5$  ，阴离子应当为  $\mathrm{AsF}_6^-$  。考虑到需要发生反应，阳离子中不应该还含有As，则反应物中参与阳离子的形成有  $\mathrm{NCI}_3$  ，形成的阳离子应当为  $\mathrm{NCI}_4^+$  。

# CHECKPOINT

1 PTS

物质A为  $\mathrm{NCl}_4\mathrm{AsF}_6$

根据题目提示“参与反应的  $\mathrm{Cl}_2$  和  $\mathrm{NCl}_3$  的摩尔比为1:2”配平方程式，则该过程的化学方程式为

$$
2 \mathrm {N C l} _ {3} + \mathrm {C l} _ {2} + 3 \mathrm {A s F} _ {5} = 2 \mathrm {N C l} _ {4} \mathrm {A s F} _ {6} + \mathrm {A s F} _ {3}
$$

# CHECKPOINT

1 PTS

形成A的化学方程式为  $2\mathrm{NCl}_3 + \mathrm{Cl}_2 + 3\mathrm{AsF}_5 = 2\mathrm{NCl}_4\mathrm{AsF}_6 + \mathrm{AsF}_3$

根据题干中信息，“升温时  $\mathbf{A}$  发生分解，产物中没有单质”，说明升温分解过程中，产物中不可能含有  $\mathrm{Cl}_2$  和  $\mathrm{N}_2$  。不妨推断元素去向：升温过程中，一定会生成氧化性气体，不可能发生 As 的还原过程，产物中 As 以  $\mathrm{AsF}_5$  存在。由于产物中不含  $\mathrm{N}_2$  ，不存在高氧化态稳定的  $\mathrm{N}$  。升温过程中，不可能发生  $\mathrm{N}$  的氧化过程，产物中  $\mathrm{N}$  以  $\mathrm{NCI}_3$  存在。考虑元素守恒且产物中不含单质，产物应当为 ClF。配平的化学方程式为

$$
\mathrm {N C l} _ {4} \mathrm {A s F} _ {6} = \mathrm {N C l} _ {3} + \mathrm {C l F} + \mathrm {A s F} _ {5}
$$

# CHECKPOINT

1 PTS

A 的分解方程式为  $\mathrm{NCl}_4\mathrm{AsF}_6 = \mathrm{NCl}_3 + \mathrm{ClF} + \mathrm{AsF}_5$

$\mathrm{NCl}_{4} \mathrm{~AsF}_{6}$  与  $\mathrm{CF}_{3} \mathrm{SCI}$  发生反应，根据题干“原有  $\mathrm{S}-\mathrm{Cl}$  键未被破坏”可以判断，反应过程中  $\mathrm{C}-\mathrm{S}$  键发生断裂。根据题干“B中阳离子对称性与  $\mathrm{NH}_{3}$  相同”，可以知道  $\mathbf{B}$  也由阴阳离子构成。观察反应物中能够形成阳离子的物质，阳离子仅能为  $\mathrm{S} 、 \mathrm{Cl}$  和  $\mathrm{CF}_{3}$  形成。考虑到对称性与  $\mathrm{NH}_{3}$  相同，阳离子应当为  $\mathrm{SCl}_{3}^{+}$  。考虑反应物中能形成阴离子的物质，不难发现阴离子仅为  $\mathrm{AsF}_{6}^{-}$  。

2 \mathrm {N C l} _ {4} \mathrm {A s F} _ {6} + \mathrm {C F} _ {3} \mathrm {S C l} = \mathrm {S C l} _ {3} \mathrm {A s F} _ {6} + \mathrm {C F} _ {4} + 2 \mathrm {N C l} _ {3} + \mathrm {A s F} _ {5}
$$
