# 题目

将  $\mathrm{[Cu(CH_3CN)_4](PF_6)}$  、  $\mathrm{Na(S_2CNR_2)}$  和  $\left[\left({}^{\mathrm{n}}\mathrm{C}_4\mathrm{H}_9\right)_4\mathrm{N}\right](\mathrm{BH}_4)$  按计量比混合溶于乙腈，在氮气保护下室温搅拌反应1小时，蒸干溶剂，用氯仿重结晶得到黄色固体。该黄色固体为  $\mathrm{Cu(I)}$  的多核簇合物，化学式为  $\mathrm{[Cu_8H(S_2CNR_2)_x](PF_6)_y}$ ，氢负离子被包在簇中；元素分析表明，该簇合物中各元素的质量分数为：C,  $23.24\%$  ；H,  $4.14\%$  ；N,  $5.64\%$

下面说法正确的是：

1. R是甲基  
2. 在阳离子中，  $\mathrm{Cu}$  共有两种环境(比例为1:1)，其中一种  $\mathrm{Cu}$  与6个  $\mathrm{Cu}$  连接，另一种与3个  $\mathrm{Cu}$  连接。则  $\mathrm{Cu}$  构成四帽四面体  
3. 在阳离子中，每个  $\mathrm{Cu}$  均被3个S配位，则每个S被2个Cu配位  
4.  $\mathrm{[Cu(CH_3CN)_4](PF_6)}$  、  $\mathrm{Cu(S_2CNR_2)_2}$  和  $\left[\left({}^{n}\mathrm{C}_{4}\mathrm{H}_{9}\right)_{4}\mathrm{N}\right]\left(\mathrm{BH}_{4}\right)$  在乙腈中按计量比混合反应也可得到该簇合物，该反应方程式（系数配平为最简整数比，方程式中物种需要反映真实结构存在形式）的化学计量数之和为41

# 答案

正确答案: K

# 详细解析

配体含碳数目为：  $14.01 / 5.64\% \times 23.24\% / 12.01 = 4.81$  ，R含2个碳

配体中含氢远大于配体外的氢数，约为：  $14.01 / 5.64\% \times 4.14\% / 1.008 = 10.2$  ，即R含5个氢

所以  $\mathbf{R} = \mathbf{Et}$  ，说法1错误

解得：  $x = 6,y = 1$
