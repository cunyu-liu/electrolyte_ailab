# DeepSeek API 配置说明

## 问题说明

您说得对！当前系统默认使用的是备用解析方案，而不是真正的DeepSeek AI解析。为了获得更准确的需求解析效果，需要配置DeepSeek API密钥。

## 配置步骤

### 1. 获取DeepSeek API密钥

1. 访问 [DeepSeek官网](https://www.deepseek.com/)
2. 注册/登录账户
3. 进入API管理页面
4. 创建新的API密钥

### 2. 配置环境变量

在 `frontend/.env` 文件中添加您的API密钥：

```bash
# DeepSeek API Configuration
REACT_APP_DEEPSEEK_API_KEY=sk-23734b4fae6c43febaea63d21ad747a2
```

### 3. 重启开发服务器

```bash
# 停止当前服务器
# 然后重新启动
npm start
```

## 对比效果

### 当前备用解析方案
- ✅ **优点**：无需API密钥，离线可用
- ❌ **缺点**：解析准确率较低，基于简单关键词匹配

### DeepSeek AI解析方案
- ✅ **优点**：解析准确率高，支持复杂语义理解
- ❌ **缺点**：需要API密钥，联网调用

## 解析能力对比

| 功能 | 备用解析 | DeepSeek AI解析 |
|------|----------|----------------|
| 正负极材料识别 | 基于关键词匹配 | 智能语义理解 |
| 性能参数提取 | 简单数值匹配 | 上下文相关解析 |
| 相对描述理解 | 有限支持 | 完整支持 |
| 置信度评估 | 固定值 | 动态计算 |
| 材料名称提取 | 基础匹配 | 高精度提取 |

## 当前改进内容

即使没有DeepSeek API，我们也已大幅改进了备用解析方案：

1. **增强材料识别**：支持更多材料名称（NCM811、NCA、LFP、Si/C等）
2. **智能体系判断**：能识别"全电池"系统类型
3. **动态基准值**：根据应用场景设定合理的性能参数基准
4. **相对描述理解**：支持"高"、"中"、"低"等相对描述
5. **完整both_materials结构**：始终提供正负极材料字段

## 注意事项

- API密钥请妥善保管，不要提交到版本控制系统
- 如果不需要AI解析功能，删除.env文件中的API密钥即可回退到备用解析
- 备用解析方案已经过优化，可以满足基本使用需求
