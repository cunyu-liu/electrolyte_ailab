# 分子生成代码库技术介绍

## 项目概述

本项目是一个专注于分子生成和分析的计算化学工具包，特别针对电解液应用中的有机分子设计。项目集成了多种分子生成算法、异构体分析工具和数据处理管道，为电解液分子设计提供全面的计算支持。

## 核心架构

项目采用模块化架构，主要分为三个功能模块：

### 1. SURGE分子生成模块 (`surge/`)

#### 核心算法
- **SURGE算法**: 基于《Algorithm for exhaustive generation of stereoisomers consistent with a given molecular formula》实现的分子结构生成算法
- **图论基础**: 使用图论方法表示分子结构，原子作为节点，化学键作为边
- **立体化学**: 支持手性中心和双键立体化学的完整枚举

#### 技术实现
- **C++核心**: 高性能的算法实现，包含73KB的主要算法代码
- **Python接口**: 通过Python脚本调用C++核心功能
- **依赖库**: 集成libnauty库用于图自同构和标准标记

#### 主要文件
- `surge/src/surge.c`: 主要的SURGE算法实现 (73KB)
- `surge/src/geng.c`: 图生成功能 (74KB)
- `surge/src/planarity.c`: 平面性检测算法 (296KB)
- `surge/usesurge/cmdline_python_gyc/test_surge.py`: Python接口示例

### 2. 异构体分析模块 (`isomers/`)

#### 功能特性
- **异构体生成**: 基于分子式生成所有可能的结构异构体
- **图论分析**: 使用NetworkX进行分子图的拓扑分析
- **化学验证**: 集成RDKit进行化学合理性和价态验证
- **周期检测**: 实现复杂的环系统检测和分析

#### 核心算法
```python
# 键序计算
def degree_count(G, n):
    i = 0
    for adj_node in G[n]:
        i += G[n][adj_node]["bond"]
    return i

# 周期检测
def has_cycle(graph):
    try:
        nx.find_cycle(graph)
        return True
    except nx.exception.NetworkXNoCycle:
        return False
```

- `analyze_results.py`: 结果分析和原子计数

### 3. 数据比较与验证模块 (`compare/`)

#### 数据处理能力
- **大规模数据处理**: 支持GB级别的化学数据库处理
- **内存优化**: 使用分块处理技术处理大型数据集
- **并行计算**: 支持多核并行SMILES标准化

#### 核心功能
- **数据库验证**: 与GDB等标准化学数据库进行比对验证
- **SMILES标准化**: 标准化分子表示式
- **重复检测**: 识别和去除重复分子结构

#### 关键算法
```python
def build_smiles_index(smi_file):
    """生成器函数逐块读取大文件"""
    chunk_size = 1_000_000  # 每次读取1M行
    current_chunk = set()

    with open(smi_file, 'r') as f:
        for i, line in enumerate(f):
            current_chunk.add(line.strip())
            if (i+1) % chunk_size == 0:
                yield current_chunk
                current_chunk = set()
        if current_chunk:
            yield current_chunk
```

### SURGE算法核心
1. **图初始化**: 基于分子式创建原子节点

4. **平面性检测**: 确保分子结构的图平面性

```python
def get_formular(num_list, symbol_list):
    a_formula = ''
    for idx, a_num in enumerate(num_list):
        if a_num == 0:
            continue
        if a_num == 1:
            a_formula = a_formula + symbol_list[idx]
        else:
            a_formula = a_formula + symbol_list[idx] + str(a_num)
    return a_formula
```
